# 酢昆布 (Sukombu)

**ScombZ 非公式 3rd パーティークライアント**

## ✨ 主な特徴

### 🎨 Material 3 Design

Material 3を採用。
~~完全な自己満。~~
### 📱 課題ウィジェット

ホーム画面に設置できる課題ウィジェット。
アプリを開かなくても、直近の締め切り課題を確認できます。
### 🏫 単位数の表示

授業詳細画面にて、その科目の単位数を表示。
公式APIに含まれているのに、公式が使用していない原因は不明。
### 🚀 公式API利用

スクレイピングではなく、公式アプリと同じAPIを~~勝手に~~使用して通信を行っています。



## 📥 インストール方法

1.  このリポジトリの [Releases](../../releases) ページへアクセスします。
2.  最新の `.apk` ファイルをダウンロードします。
3.  Android端末でファイルを開き、インストールしてください（提供元不明のアプリのインストール許可が必要な場合があります）。
